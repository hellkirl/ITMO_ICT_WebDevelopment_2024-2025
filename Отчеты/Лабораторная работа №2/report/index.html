    Отчет

    Введение

    В рамках данного проекта была разработо веб-приложение для управления отелями.

    База данных

    Схема базы данных

    Таблица: users

    Поле Тип Описание
    id SERIAL Уникальный идентификатор
    username VARCHAR(50) Имя пользователя
    email VARCHAR(100) Электронная почта (уникальная)
    password VARCHAR(255) Захешированный пароль
    role VARCHAR(20) Роль пользователя (admin/user)
    created_at TIMESTAMP Дата и время создания записи

    Таблица: hotels

    Поле Тип Описание
    id SERIAL Уникальный идентификатор
    name VARCHAR(100) Название отеля
    address VARCHAR(255) Адрес
    city VARCHAR(100) Город
    country VARCHAR(100) Страна
    phone VARCHAR(20) Контактный номер
    email VARCHAR(100) Электронная почта
    image VARCHAR(255) URL изображения отеля
    created_at TIMESTAMP Дата и время создания записи

    Таблица: rooms

    Поле Тип Описание
    id SERIAL Уникальный идентификатор
    hotel_id INTEGER Внешний ключ на hotels.id
    number VARCHAR(10) Номер комнаты
    type VARCHAR(50) Тип комнаты (например, Single)
    price DECIMAL(10,2) Стоимость за ночь
    created_at TIMESTAMP Дата и время создания записи

    Таблица: reservations

    Поле Тип Описание
    id SERIAL Уникальный идентификатор
    user_id INTEGER Внешний ключ на users.id
    room_id INTEGER Внешний ключ на rooms.id
    check_in DATE Дата заезда
    check_out DATE Дата выезда
    status VARCHAR(20) Статус резервации (reserved/checked_in/checked_out)
    created_at TIMESTAMP Дата и время создания записи

    Таблица: reviews

    Поле Тип Описание
    id SERIAL Уникальный идентификатор
    reservation_id INTEGER Внешний ключ на reservations.id
    rating INTEGER Рейтинг (1-5)
    comment TEXT Текст отзыва
    created_at TIMESTAMP Дата и время создания записи
